## Getting Started

* Clone this repo <https://github.com/ksigei/rails_blog.git>

    ```bash
    git clone https://github.com/ksigei/rails_blog.git
    ```

* Navigate to rails_blog folder/directory

    ```bash
    cd rails_blog
    ```

* Install gems

    ```bash
    bundle install
    ```

* Create your postgres database

    ```bash
    rails db:create
    ```
    ```bash
    rails db:migrate
    ```
* Run the project

    ```bash
    rails s
    ```
